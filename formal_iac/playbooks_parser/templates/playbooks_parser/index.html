Formal Verification and Static Analysis of Infrastructure as Code Environments

Code linting, compilers, static and dynamic analysis...nowadays software development makes use of several tools that by formal methods, aim at improving the security and reliability of our systems. However, in this new paradigm where Infrastructure as a Code and configuration management has irrupted in modern organizations, the formal verification field has not been as developed as in its pure software homologous.
